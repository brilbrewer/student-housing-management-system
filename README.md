About the Project

RentMatch is a full-stack web application designed to streamline the rental listing and tenant management process. The platform allows users to browse available properties, submit interest forms, and manage renter and property data through a dynamic, database-driven system.

This project highlights my ability to design and build a complete CRUD-based web application, combining front-end UI design with back-end database logic.
